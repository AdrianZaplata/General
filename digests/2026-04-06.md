**Claude Code Daily** — 2026-04-06

**TL;DR:** Anthropic accidentally leaked ~512K lines of Claude Code's TypeScript source code via an npm source map — the biggest story of the week. Separately, Anthropic cut off Claude Pro/Max subscriptions from third-party agentic tools, sparking community backlash and rate-limit frustration.

---

**Big News**
- **[Source code leaked via npm](https://build.ms/2026/4/1/the-claude-code-leak/)** — A 59.8 MB source map accidentally shipped in an npm release, exposing ~512K lines of unobfuscated TypeScript across ~1,900 files. [Fact check confirms](https://leadstories.com/hoax-alert/2026/04/fact-check-leak-of-anthropics-claude-code-source-code-was-not-an-april-fools-prank.html) it was NOT an April Fools stunt. Anthropic confirmed no customer data was exposed.
- **Hidden features exposed by leak** — Code reveals "Kairos" (background idle summarizer), "Undercover Mode" (AI-undisclosed OSS contributions), and a Tamagotchi easter egg. [HN discussion](https://news.ycombinator.com/item?id=47586778) digging into the details.
- **[Third-party agentic tool subscriptions blocked](https://www.dataworldbank.net/2026/04/04/anthropic-cuts-off-the-ability-to-use-claude-subscriptions-with-openclaw-and-third-party-ai-agents/)** — Effective April 4, Claude Pro/Max can no longer power tools like OpenClaw. Anthropic is offering a one-time credit equal to your monthly plan (valid through April 17) plus up to 30% off pre-purchased extra usage bundles.
- **Model deprecations incoming** — Claude Haiku 3 retires April 19. The 1M token context beta for Sonnet 4.5/4 retires April 30. Max tokens on Message Batches API raised to 300K for Opus 4.6 and Sonnet 4.6.

---

**Power User Tips**
- **[/powerup command](https://claudelab.net/en/articles/claude-code/claude-code-powerup-command-guide)** — New in v2.1.90, `/powerup` launches interactive in-terminal animated tutorials covering compaction, context resets, and session management. Run it if you haven't already.
- **Parallel branch sessions** — Run 4–5 Claude Code sessions on separate git branches simultaneously, each with its own plan. Compare results before merging. Fastest way to explore competing implementations.
- **Personal skills in `~/.claude/skills/`** — Skills placed here are available across all your projects, not just the current repo.
