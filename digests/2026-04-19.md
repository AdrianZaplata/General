**Power User Tips**
- **Plan before you code.** Anthropic's own testing shows unguided Claude attempts succeed ~33% of the time; adding an explicit plan step dramatically improves outcomes. Never let Claude write a line until the plan is approved.
- **Run parallel instances.** Five Claude Code terminals × five tasks compresses a 5-hour sequential job to 1–2 hours. Each instance gets its own context — no cross-contamination.
- **Use subagents for isolation.** Drop specialized agent configs in `.claude/agents/` so Claude delegates file-heavy or focused subtasks without polluting your main conversation context.
- **Try Ultraplan for big projects.** (Week 15, April 6–10) Draft a plan from the CLI, refine it in the web editor, then run it remotely or pull it local. First run auto-provisions a cloud environment.
- **[CodeBurn](https://news.ycombinator.com/item?id=47759035)** (Show HN, ~April 17) — New open-source tool reads your Claude Code JSONL session transcripts and classifies token spend by task type. Useful for spotting what's eating your context budget.
