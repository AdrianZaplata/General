Max 20x users reported draining quotas in ~70 minutes; a GitHub issue traced it to a regression in v2.1.89. ([The Register](https://www.theregister.com/2026/03/31/anthropic_claude_code_limits/)) / ([devclass](https://www.devclass.com/ai-ml/2026/04/01/anthropic-admits-claude-code-users-hitting-usage-limits-way-faster-than-expected/5213575))

- **1M token context window retirement announced** — Claude Sonnet 4.5 and Sonnet 4 will drop their 1M-context beta on **April 30**. Migrate to Sonnet 4.6 or Opus 4.6, which include 1M context at standard pricing with no beta header needed. ([Anthropic API Docs](https://platform.claude.com/docs/en/release-notes/overview))

- **New `/powerup` command** — Interactive in-terminal lessons for Claude Code features, complete with animated demos. Handy for onboarding teammates or rediscovering features you've forgotten. ([releasebot.io](https://releasebot.io/updates/anthropic/claude-code))

- **Hooks beat CLAUDE.md for critical automation** — Unlike CLAUDE.md instructions (which are advisory), hooks execute shell scripts deterministically at specific workflow points. If something *must* happen every time, put it in a hook, not a doc.
