**Claude Code Daily** — 2026-04-09

**TL;DR:** AMD's AI director published data-backed evidence that Claude Code got significantly worse after the February update, sparking a heated HN thread. A critical security bypass vulnerability was patched April 6 — but InfoWorld reports many users are still exposed on older versions. Meanwhile, the latest releases bring named subagents, PowerShell support, and MCP result-size controls.

---

**Big News**

- **AMD's AI director: "Claude Code cannot be trusted for complex engineering"** — Stella Laurenzo published findings from 6,800+ coding sessions, 235K tool calls, and 18K reasoning blocks showing Claude Code got ~67% less effective at complex tasks since the February 2026 update. Her team stopped using it for kernel-level and hardware debugging work. Anthropic hasn't formally responded yet. [The Register](https://www.theregister.com/2026/04/06/anthropic_claude_code_dumber_lazier_amd_ai_director/) · [TechRadar](https://www.techradar.com/pro/claude-cannot-be-trusted-to-perform-complex-engineering-tasks-amd-ai-head-slams-anthropics-coding-tool-after-months-of-frustration/)

- **Critical security vulnerability patched — but many users still exposed** — Anthropic patched a command-parser bug (April 6) that silently bypassed developer-configured deny rules, potentially allowing malicious prompts to execute blocked commands. [InfoWorld](https://www.infoworld.com/article/4154199/claude-code-is-still-vulnerable-to-an-attack-anthropic-has-already-fixed.html) reports users on older versions remain at risk — **update your install now**. [SecurityWeek](https://www.securityweek.com/critical-vulnerability-in-claude-code-emerges-days-after-source-leak/)

- **New release: named subagents, PowerShell support, alt-screen rendering** — The latest builds add named subagents in `@` mentions (clearer multi-agent workflows), flicker-free alt-screen rendering for large outputs, and broader PowerShell permission support on Windows. Long-session stability also significantly improved. [Releasebot](https://releasebot.io/updates/anthropic/claude-code)

**Power User Tips**

- **MCP result size now configurable (v2.1.91)** — You can cap MCP tool output at up to 500,000 characters per result via `_meta["anthropic/maxResultSizeChars"]`. Prevents runaway context bloat from verbose tool responses.

- **`/cost` got smarter** — Now shows per-model and cache-hit cost breakdowns for Max subscribers. If you're running parallel sessions, this finally lets you see exactly where tokens are being burned. [Daily 1 Bite](https://daily1bite.com/en/blog/ai-tutorial/claude-code-april-2026-update)

- **Bedrock setup is now interactive** — The `/powerup` wizard includes a new guided AWS Bedrock setup flow (authentication → region → credential check → model pinning). Much friendlier than manual config for enterprise teams. [Claude Lab](https://claudelab.net/en/articles/claude-code/claude-code-powerup-command-guide)
