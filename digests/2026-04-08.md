**Claude Code Daily** — 2026-04-08

**TL;DR:** Anthropic dropped a bombshell yesterday: Claude Mythos — their most powerful model ever — is real, benchmarks at 93.9% SWE-bench, and won't be released publicly due to cybersecurity risks. Meanwhile, the accidental npm source-code leak from last week is still dominating discourse, and a controversial subscription policy change is pushing users toward alternatives.

**Big News**

- **Claude Mythos Preview announced (April 7)** — Anthropic's most capable model yet scores 93.9% on SWE-bench Verified and autonomously discovered thousands of zero-days across every major OS and browser. It's being kept off public release; instead, Anthropic launched [Project Glasswing](https://techcrunch.com/2026/04/07/anthropic-mythos-ai-model-preview-security/) to give 40 partner orgs (AWS, Apple, Google, Microsoft, Nvidia…) access for *defensive* security work only. [TechCrunch](https://techcrunch.com/2026/04/07/anthropic-mythos-ai-model-preview-security/) · [Fortune](https://fortune.com/2026/04/07/anthropic-claude-mythos-model-project-glasswing-cybersecurity/)

- **Source code leak aftermath still viral** — Anthropic accidentally shipped a 59.8 MB `.map` file in `@anthropic-ai/claude-code` v2.1.88, exposing 512,000+ lines of TypeScript. The community found 44 unreleased feature flags, internal model codenames, multi-agent orchestration internals, and a hidden Claude Buddy (Tamagotchi-style terminal pet). Anthropic called it a packaging error; the original post hit **16M views on X**. [VentureBeat](https://venturebeat.com/technology/claude-codes-source-code-appears-to-have-leaked-heres-what-we-know/) · [HN thread](https://news.ycombinator.com/item?id=47609294)

- **Subscriptions no longer cover third-party tools (as of April 4)** — Claude Code subscriptions can no longer be used with harnesses like OpenClaw. Pay-as-you-go billing is now required for third-party integrations. Community reaction has been sharply negative. [HN](https://news.ycombinator.com/item?id=47633396) · [IndexBox coverage](https://www.indexbox.io/blog/anthropic-changes-claude-code-subscription-for-third-party-tools-in-2026/)

- **Message Batches API: max_tokens raised to 300k** — Claude Opus 4.6 and Sonnet 4.6 now support up to 300,000 output tokens per request on the Batches API. Big win for long-form generation pipelines.

**Power User Tips**

- **`/powerup` — interactive in-terminal learning** — New in v2.1.90, this command launches animated, interactive lessons covering context management, hooks, and more. Best onboarding path for new Claude Code users without leaving the terminal. [Guide](https://claudelab.net/en/articles/claude-code/claude-code-powerup-command-guide)

- **Boris Cherny's actual workflow (Claude Code's creator)** — His CLAUDE.md stays ~100 lines / 2,500 tokens. He runs 10–15 parallel sessions (terminal tabs + web + phone), each in its own git worktree to prevent collisions. Always starts in Plan Mode: one Claude drafts, a second reviews "as a staff engineer." [Full breakdown](https://mindwiredai.com/2026/03/25/claude-code-creator-workflow-claudemd/)

- **`lessons.md` self-improvement pattern** — Every time Claude makes a mistake and gets corrected, have it append a rule to `lessons.md`. Over time Claude teaches itself the quirks of your specific project. Simple, high-leverage habit.
