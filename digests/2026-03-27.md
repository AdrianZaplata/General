**Official Updates**

- **[v2.1.85 released (Mar 26)](https://github.com/anthropics/claude-code/releases)** — Adds conditional `if` field for hooks (filter when they fire using permission rule syntax), timestamp markers in `/loop` and `CronCreate` transcripts, MCP OAuth RFC 9728 compliance, and fixes `/compact` crashing on large conversations. Also replaces WASM yoga-layout with pure TypeScript — noticeably smoother scrolling on big transcripts.

- **[v2.1.84 released (Mar 26)](https://github.com/anthropics/claude-code/releases)** — PowerShell tool for Windows lands as opt-in preview. Adds `TaskCreated` hook, deep link support up to 5,000 chars, and fixes voice push-to-talk leaking characters into text input.

- **[Auto Mode research preview](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)** — Anthropic's new autonomous decision-making mode lets Claude judge which actions are safe without asking permission. Currently Sonnet 4.6/Opus 4.6 only; Anthropic recommends isolated environments. Rolling out to Enterprise and API users imminently.

- **[Claude Code Channels — Telegram & Discord](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)** — Ship tasks to Claude Code from your phone via Telegram or Discord. Configure with `/telegram:configure`, pair via security code, and your phone becomes a remote control for your dev environment. Shipped March 20.

- **[Usage quota doubled through March 27](https://medium.com/@william.couturier/claude-code-in-march-2026-the-economics-of-the-quota-792449b63edb)** — From March 13–27, Anthropic doubled quotas on all plans outside the 8 AM–2 PM ET window. Today is the last day — take advantage of off-peak hours.
