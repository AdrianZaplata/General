**Power User Tips**

- **`/powerup` is the onboarding resource you've been missing** — Released April 1, this in-terminal interactive tutorial system walks through Claude Code's most powerful features via animated demos without leaving your shell. If you're still learning the tool or onboarding a teammate, this is now the fastest path. [ClaudeLab guide](https://claudelab.net/en/articles/claude-code/claude-code-powerup-command-guide)

- **Subagents keep your main context lean** — Appending "use subagents" to any heavy task spins up isolated agents with their own context windows. They do the exploration, then report back a compressed summary. Your main session stays clean and fast, especially useful on long-running refactors. [Best practices docs](https://code.claude.com/docs/en/best-practices)

- **`/loop` for recurring tasks** — `/loop 5m /babysit` auto-addresses review comments on a schedule; `/loop 30m /slack-feedback` can auto-draft PRs. Set it and come back. Pairs well with subagents so you're not burning main-session limits on polling work.
