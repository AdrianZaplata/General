**Big News**
- **[Claude Mythos accidentally leaked](https://fortune.com/2026/03/26/anthropic-says-testing-mythos-powerful-new-ai-model-after-data-leak-reveals-its-existence-step-change-in-capabilities/)** — A CMS misconfiguration on March 26–27 exposed ~3,000 unpublished Anthropic assets, revealing a new model tier ("Capybara/Mythos") that dramatically outperforms Opus 4.6 in coding, reasoning, and cybersecurity. Anthropic confirmed it's real, calling it "the most capable model we've built" — but it won't be publicly released soon due to extreme cost and safety concerns. ([The Decoder](https://the-decoder.com/anthropic-leak-reveals-new-model-claude-mythos-with-dramatically-higher-scores-on-tests-than-any-previous-model/), [Futurism](https://futurism.com/artificial-intelligence/anthropic-step-change-new-model-claude-mythos))
- **[Claude Code Auto Mode](https://blog.laozhang.ai/en/posts/claude-code-auto-mode)** (research preview, Teams) — Uses a Sonnet 4.6 safety classifier to auto-approve low-risk actions and block/challenge higher-blast-radius ones (production deploys, force-pushes to main). Anthropic's data: users already approve 93% of prompts, so this cuts real busywork. ([TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/))
- **[Claude Desktop Dispatch bug (today)](https://status.claude.com/)** — Claude 1.1.9310 has a known issue where Dispatch sessions stop returning replies. Workaround: use standard Cowork sessions instead.
- **Usage promo ended March 28** — The doubled off-peak quotas (all plans) that ran since March 13 are now over. Expect tighter limits, especially during 5–11am PT peak hours.

**Power User Tips**
- **[How I'm Productive with Claude Code](https://news.ycombinator.com/item?id=47494890)** (HN, 2 days ago) — Top tips: spin up multiple agents to implement the same requirement independently and combine the best ideas; point an agent at focused documentation for narrowly scoped questions; have agents abstract review comments into reusable checklists.
- **[Claude Code Cheat Sheet](https://news.ycombinator.com/item?id=47495527)** trending on HN — Covers CLI flags, key bindings, and model-switching shortcuts worth bookmarking.
- **OpusPlan workflow**: `/model opusplan` → Shift+Tab×2 to plan with Opus, then Shift+Tab to execute with Sonnet. Optimizes cost while keeping quality on multi-file tasks.
- **Hooks > CLAUDE.md**: CLAUDE.md is advisory (~80% compliance). Hooks are deterministic (100%). If it must always run — linting, security checks — make it a hook.
