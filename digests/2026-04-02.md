**Claude Code Daily** — 2026-04-02

**TL;DR:** Anthropic accidentally leaked 512,000 lines of Claude Code source code via a stray npm source map on March 31, instantly becoming the biggest AI leak of 2026. The fallout includes a supply chain attack, mass DMCA takedowns (later reversed), and the fastest-growing GitHub repo in history — plus major unreleased features now publicly known.

---

**Big News**
- **[The Claude Code source leak](https://venturebeat.com/technology/claude-codes-source-code-appears-to-have-leaked-heres-what-we-know/)** — npm package v2.1.88 shipped with a 59.8 MB source map pointing to a ZIP of ~1,900 TypeScript files on Anthropic's Cloudflare R2 bucket. Root cause: Bun generates source maps by default; someone forgot to add `*.map` to `.npmignore`. Anthropic confirmed: "a release packaging issue caused by human error, not a security breach."
- **[DMCA chaos + viral explosion](https://techcrunch.com/2026/04/01/anthropic-took-down-thousands-of-github-repos-trying-to-yank-its-leaked-source-code-a-move-the-company-says-was-an-accident/)** — A security researcher's tweet hit 21-30M views. One fork reached 32,600 stars before DMCA concerns. Anthropic filed ~8,100 takedowns (accidentally hitting legitimate forks of their own public repos); most were later reversed. A clean-room Rust rewrite "claw-code" hit 100K stars in one day — fastest-growing repo in GitHub history.
- **[What the leak revealed](https://tech.yahoo.com/ai/claude/articles/claude-code-leak-suggests-anthropic-150107499.html)** — Key unreleased features exposed: **KAIROS/Proactive Mode** (always-on mode where Claude watches and acts without waiting for prompts), references to **Claude Capybara v2** model, a crypto/autonomous agent payment system, and a "Dreaming" memory consolidation process.
- **[Supply chain attack warning](https://www.bleepingcomputer.com/news/artificial-intelligence/claude-code-source-code-accidentally-leaked-in-npm-package/)** — Users who `npm install`-ed Claude Code on March 31 between 00:21–03:29 UTC may have pulled a trojanized version containing a cross-platform RAT. If you installed in that window, audit your system. Attackers also registered typosquat npm packages targeting devs trying to compile the leaked source.
- **Claude Code v2.1.89 released (April 1)** — Real features alongside the April Fools `/buddy` command: PreToolUse hooks can now **defer** permission decisions (big for automation), named subagents appear in `@` mention typeahead, and `CLAUDE_CODE_NO_FLICKER=1` graduated from beta.
