**Claude Code Daily** — 2026-04-22

**TL;DR:** Anthropic quietly tested pulling Claude Code from the $20/month Pro plan — locking it to $100/month Max — before reversing course within hours after widespread backlash. Meanwhile, Claude Design (Anthropic's new AI visual tool) is quietly becoming the week's other big story, having launched Friday and already sending Figma stock down 7%.

---

**Big News**
- **[Anthropic briefly drops Claude Code from Pro plan](https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/)** — Pricing pages were quietly updated to show an "✗" for Claude Code under the $20 Pro tier, with full access requiring the $100–200/month Max plan. Anthropic's head of growth called it a "~2% test on new prosumer signups." But support docs were *also* rewritten ("Using Claude Code with your Max plan"), undercutting the "small test" framing. Pages appear to have been reverted. [Simon Willison's breakdown](https://simonwillison.net/2026/Apr/22/claude-code-confusion/) is the clearest read on the confusion.
- **[Claude Design launches as research preview](https://www.anthropic.com/news/claude-design-anthropic-labs)** — Anthropic Labs' new tool lets you describe a design and Claude builds it — prototypes, slides, one-pagers, decks. Powered by Opus 4.7. Comment inline, tweak spacing/color live, export to PDF, PPTX, or HTML. Available now to Pro/Max/Team/Enterprise at no extra charge. [TechCrunch](https://techcrunch.com/2026/04/17/anthropic-launches-claude-design-a-new-product-for-creating-quick-visuals/) has the full breakdown.
- **Claude Opus 4.7 now generally available** — Better software engineering performance, higher-resolution vision, and it now powers Claude Design. Upgrade path: `claude-opus-4-7` in the API.
