**Claude Code Daily** — 2026-04-11

**TL;DR:** The Claude Code source leak has evolved into an active security threat — hackers are using the leaked code to distribute malware via GitHub. Separately, Anthropic escalated its third-party harness crackdown by temporarily banning OpenClaw's creator from Claude entirely, and community frustration over rate limits and quality regressions is peaking on HN.

---

**Big News**

- **Leaked source code now spreading malware** — Threat actors are exploiting the March 31 source leak to distribute Vidar stealer and GhostSocks proxy malware, publishing them as "leaked Anthropic tools" on GitHub Releases. If you downloaded any unofficial Claude Code packages this week, audit your system. [GBHackers](https://gbhackers.com/claude-code-leak/) · [LeadStories fact check](https://leadstories.com/hoax-alert/2026/04/fact-check-leak-of-anthropics-claude-code-source-code-was-not-an-april-fools-prank.html)

- **Anthropic temporarily banned OpenClaw's creator** — Following the April 4 policy that blocked Claude Pro/Max subscriptions from working with third-party harnesses like OpenClaw, Anthropic went further on April 10 and temporarily revoked API access for OpenClaw's creator. The creator (who had already moved to OpenAI in February) called it "a betrayal of open-source developers." Community reaction is heated. [TechCrunch](https://techcrunch.com/2026/04/10/anthropic-temporarily-banned-openclaws-creator-from-accessing-claude/) · [VentureBeat](https://venturebeat.com/technology/anthropic-cuts-off-the-ability-to-use-claude-subscriptions-with-openclaw-and) · [HN thread](https://news.ycombinator.com/item?id=47633396)

- **Rate limiting hitting hard — users asking "what's next?"** — An Ask HN thread ("What are you moving on to now that Claude Code is so rate limited?") is gaining traction, with users reporting they hit 20% of weekly limits in ~2 hours of real work. Coming on top of quality regressions and outages, this is becoming a retention risk for Anthropic. [HN](https://news.ycombinator.com/item?id=47626833)

**Power User Tips**

- **`/powerup` — interactive lessons built in** — New in the latest release: `/powerup` launches animated, interactive lessons teaching Claude Code features from within the CLI itself. Good onboarding shortcut to share with teammates who are still learning the tool.

- **500K character MCP tool results** — You can now annotate MCP tool results with `_meta: { maxChars: 500000 }` to unlock up to 500K characters per result (up from the previous default). Useful for large file reads, database dumps, or long test outputs piped through MCP.

- **Focus view for distraction-free prompting** — Toggle Focus view with `Ctrl+O` (in `NO_FLICKER` mode) to strip the UI down to just your prompt, a one-line tool summary with edit diffstats, and the final response. Useful for long agentic runs where you want signal, not noise.
