**Claude Code Daily** — 2026-04-15

**TL;DR:** Anthropic shipped Routines — scheduled, cloud-hosted automations that let Claude work overnight without your laptop on. Meanwhile rate-limiting rage is boiling over on HN and GitHub, with developers publicly asking what to switch to.

---

**Big News**

- **Claude Code Routines: automated tasks that run while you sleep** — Launched April 14 as a research preview, Routines are saved Claude Code configurations (prompt + repo + connectors + trigger) that run on Anthropic's cloud infrastructure. Three trigger types: cron schedule, API call, or GitHub event. Example use case: pull the top bug from Linear at 2am, attempt a fix, open a draft PR before anyone wakes up. Daily caps: Pro gets 5 runs, Max 15, Team/Enterprise 25. [9to5Mac](https://9to5mac.com/2026/04/14/anthropic-adds-repeatable-routines-feature-to-claude-code-heres-how-it-works/) · [SiliconANGLE](https://siliconangle.com/2026/04/14/anthropics-claude-code-gets-automated-routines-desktop-makeover/) · [Claude Docs](https://code.claude.com/docs/en/routines)

**Power User Tips**

- **`/loop` for local recurring tasks, Routines for cloud tasks** — `/loop` schedules a recurring prompt locally for up to 3 days (auto-addressing reviews, rebasing, shepherding PRs). Routines run on Anthropic's infra even when your machine is off. Know which to reach for: local + short-lived = `/loop`; overnight + multi-repo = Routines.

- **`/powerup` is an interactive in-terminal tutorial** — Released April 1 in v2.1.90, it walks you through Claude Code's most powerful features via animated in-terminal demos. If you haven't run it yet, it's the fastest way to find workflows you're missing.

**Trending Discussions**

- **Rate-limit backlash reaches HN boiling point** — "Ask HN: What are you moving on to now that Claude Code is so rate limited?" is drawing hundreds of comments, with users reporting they hit 20% of their weekly limit in ~2 hours of real work. Gemini CLI and Cursor are emerging as the most-cited alternatives. [HN](https://news.ycombinator.com/item?id=47626833)

- **GitHub issue: "Claude Code is unusable for complex engineering tasks"** — Following AMD's Stella Laurenzo's April 2 data-backed regression report, a separate GitHub issue is now gaining HN traction, detailing failures on multi-file refactors, wrong-file edits, and reasoning loops. Anthropic's Boris Cherny maintains the `redact-thinking` header change is UI-only. Debate continues. [HN](https://news.ycombinator.com/item?id=47660925) · [GitHub](https://github.com/anthropics/claude-code/issues/42796)

- **The Register on Routines: "mildly clever cron jobs"** — The Register published a characteristically skeptical take, calling Routines "a YAML trigger with a Claude budget" and questioning whether cloud-hosted automation justifies the run caps. Worth reading for the counter-narrative. [The Register](https://www.theregister.com/2026/04/14/claude_code_routines)
