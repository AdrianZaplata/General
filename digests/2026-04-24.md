**Big News**

[Simon Willison](https://simonwillison.net/2026/Apr/22/claude-code-confusion/) called it "a whole lot of people getting scared or angry that a service they relied on was about to be rug-pulled." Available to Pro, Max, Team, and Enterprise subscribers.
- **[Anthropic formally commits Claude will never show ads](https://blurbrahlab.medium.com/anthropic-says-claude-will-never-show-ads-top-10-ai-flutter-news-april-21-2026-917228184ff2)** — "Claude is a space to think": no sponsored links, no advertiser influence on responses, no product placements.

**Power User Tips**

- **Run 5 parallel Claude Code instances** — Boris Cherny (Claude Code's creator) works across 5 terminal tabs, each with its own git worktree of the same repo. Tasks that take 5 hours sequentially compress to 1–2 hours. ([source](https://mindwiredai.com/2026/04/14/claude-code-creator-workflow-boris-cherny/))
- **CLAUDE.md is your persistent memory** — Document mistakes, style conventions, PR templates, and design guidelines. Claude reads it fresh every session. Keep it scannable in under 2 minutes.
- **Plan Mode before a single line of code** — Have Claude write a spec/design doc and approve it before coding starts. Unguided attempts succeed ~33% of the time; planned ones are dramatically higher.
- **Verification loops beat everything else** — Give Claude a way to check its own output (run tests, lint, query the API). The single highest-leverage practice for anyone using Claude Code seriously.
